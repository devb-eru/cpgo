
# 핵심 기술 융합 방안

## 1. 시스템 개념 아키텍처

본 서비스는 사용자의 모바일 클라이언트, 온디바이스 AI 및 AR 엔진, 그리고 클라우드 데이터 서버(문화체육관광부 공공데이터 포털, 영화진흥위원회, 예술위 등 연계)의 유기적인 3-Layer 아키텍처로 구성됨.

```text
+-------------------------------------------------------------+
|                     1. 모바일 클라이언트                      
|  - UI/UX: 마이 갤러리 3D 렌더러, 소셜 피드, 카드 제작 에디터      
|  - AR/카메라: ARCore/ARKit 기반 증강현실 뷰어, GPS 트래커      
+-------------------------------------------------------------+
                              |
                              v
+-------------------------------------------------------------+
|                     2. AI & AR 하이브리드 엔진               
|  - 검증 모듈: 온디바이스 OCR + 체류 시간(Dwell-Time) 모니터링
|  - AI 생성: A안 분류 필터 + 3D 가우시안 스플래팅(3DGS) 엔진 
|  - AR 투영: SLAM 및 자이로 센서 기반 가상 오브젝트 배치 엔진
+-------------------------------------------------------------+
                              |
                              v
+-------------------------------------------------------------+
|                     3. 공공데이터 및 클라우드 API            
|  - 문화데이터광장: 문화재, 미술 전시 기본 정보 데이터         
|  - 영화진흥위원회 / 예술위 / 콘진원: 영화, 소극장, 이벤트 API  
|  - 한국관광공사: 주변 관광지, 상권 위치 및 연계 루트 정보      
|  - 국민체육진흥공단: 도보 이동 트래킹 기반 피트니스 데이터   
+-------------------------------------------------------------+
```

## 2. AI DX 기술 융합 (인공지능 기반 디지털 전환)

* ~~**경량 온디바이스 컴퓨터 비전**: 모바일 기기의 AP 자원을 최대로 활용하여 인터넷이 불안정한 지방 산간 오지에서도 실시간으로 문화재를 스캔할 수 있도록 온디바이스 딥러닝 객체 검출(MobileNet, YOLO-Lite 계열) 모델을 탑재함.~~
* **문화생활 오브젝트 분류 AI 필터 (A안)**: 사용자가 촬영한 사진에서 임의의 부적절한 사물을 차단하기 위해, 온디바이스 경량 분류 모델(YOLO-Lite 또는 MobileNet 기반 커스텀 튜닝)이 실물 티켓, 팜플렛, 무대 소품, 전시 부스 등 문화생활 관련 오브젝트 여부를 1차 판단함.
* **3D 가우시안 스플래팅(3D Gaussian Splatting, 3DGS) 기반 가상 자산 생성**: 기존 NeRF(신경방사휘도장) 기술의 긴 연산 시간과 하드웨어 리소스 부담 한계를 극복하기 위해, 모바일 렌더링에 특화된 3DGS 알고리즘을 사용함. 사용자가 다각도에서 가볍게 촬영한 사진 데이터를 기반으로 단시간 내에 고화질의 3D 공간 입체 오브젝트를 변환하고 추출해 냄.
* **AI 스타일 변환 및 이미지 생성**: 사용자가 획득한 사진을 고풍스러운 수묵화, 수채화, 조각보 콜라주, 레트로 픽셀아트로 변환하기 위해 디퓨전(Diffusion) 기반의 고속 스타일 변환 알고리즘을 도입함. 클라우드 연동을 통한 최적화로 모바일 기기 내에서 3초 이내에 완성도 높은 디지털 카드를 재가공해 냄.

## 3. AR(증강현실) 문화 공간 시각화 및 인터랙티브 투영

* **공간 동시적 정합 및 위치 추적(SLAM)**: 스마트폰의 카메라 센서, 가속도계, 자이로스코프 센서를 융합하여 사용자의 3차원 공간 평면을 정확하게 스캔하고 좌표를 설정함.
* ~~**소실 부위 복원 투영**: 공간 스캔이 완료되면 문화유산 3D 모델링 원형 데이터를 실제 소외 문화재 물리적 위치 위에 정합하여 투영함. 유실된 지붕돌, 무너진 돌벽 등이 카메라 화면 너머로 완벽하게 겹쳐 보이도록 구성함.~~
* **생성된 3D 오브젝트의 공간 투영 및 마이 갤러리 렌더링**: 3DGS 기술로 생성된 소극장 무대 소품, 미술품, 팝업 굿즈 등의 3D 오브젝트를 사용자의 실제 방이나 실외 빈 공간에 AR로 자유롭게 투영하고 정합함. 사용자는 마이 갤러리 내에 이 오브젝트들을 회전, 크기 조절, 배치하여 나만의 가상 컬렉션 공간을 조립할 수 있음.
* ~~**인터랙티브 미니 퀘스트 복원**: 사용자가 단방향으로 복원된 3D 그래픽을 감상하는 것을 넘어, 터치 인터페이스를 통해 가상 기와를 얹거나, 무너진 석축을 손가락으로 드래그하여 올바른 자리에 맞추는 'AR 조립 퀘스트'를 수행함. 퍼즐을 맞추듯 복원 퀘스트를 성공시키면 해당 문화재 카드의 특별 수집 엠블럼과 마이 갤러리 보상이 해제됨.~~
* **인터랙티브 3D 공간 상호작용**: 수집한 3D 오브젝트들을 가상 공간에 배치할 때, 터치 인터페이스를 사용하여 유저가 오브젝트를 직접 조립하거나 개별 텍스처를 씌우는 미니 퀘스트를 제공함. 조립 완료 시 해당 카드의 특별 소셜 엠블럼과 마이 갤러리 추가 데코가 해제됨.

## 4. 다종 데이터 연계 아키텍처

문체부 및 산하 문화 예술 관련 기관들의 다채로운 공공 API를 매끄럽게 연결하여 초개인화 경험을 뒷받침함.

* **문화데이터 광장 (API)**: 전국 지자체 및 국가유산청이 보유한 문화유산 정보 및 국립현대미술관 등의 미술 전시 데이터를 호출하여 카드 뒷면의 기본 도슨트 텍스트 정보로 탑재함.
* **영화진흥위원회 통합전산망 API**: 상영 중인 독립 영화 목록 및 영화관 위치 정보를 실시간 호출함.
* **한국예술위원회 및 한국콘텐츠진흥원 API**: 대학로 소극장을 포함한 전국 공연 정보, 예매 일정 및 로컬 콘텐츠/팝업 행사 데이터를 연계하여 GPS 탐색 데이터베이스를 업데이트함.
* **TourAPI (한국관광공사)**: 유저가 수집 중인 소외 문화재 좌표를 기준으로 주변 맛집, 숙박 시설, 축제 정보, 도보 여행 코스를 매핑하여 실시간 이동 루트를 구성함.
* **국민체육진흥공단 API**: 걷기 트래킹 및 건강 피트니스 연동을 위한 스포츠 활동 데이터를 연동함.
